### 📌 Featured Project
#### HireFlow – Connect.Apply.Grow 

**HireFlow** is a minimal, portfolio-ready **job recruitment platform** built with **Django & Django REST Framework**, designed to showcase full-stack backend skills and production-ready practices.

#### HireFlow Features :

##### 👥 Role-Based System
- Separate dashboards for Employers and Candidates
- Controlled access based on user roles

##### 🔐 Authentication & Security
- Secure signup & login system
- Role-based authorization

##### 💼 Job Management
- Employers can create, update, and manage job posts
- Candidates can browse, filter, and apply for jobs

##### 🌐 REST API
- Full API support for jobs, applications, and users
- Built with Django REST Framework

##### 📂 File Uploads
- Resume and profile image uploads using Cloudinary

##### 📧 Email System
- Welcome emails using Celery + Redis (local)
- SMTP support for production (Render free tier)

##### 🔎 Search & Navigation
- Job filtering, search, and pagination system

##### ⚙️ Background Tasks
- Celery + Redis for async task handling

##### 🚀 Deployment Ready
- PostgreSQL (Supabase) for production database
- Static files handled via Whitenoise 
- Deployed on Render

##### 🧹 Clean Architecture
- Organized settings, signals, and task structure
- Scalable backend design
